# frontend-web

Aplicacion web del proyecto Risk basada en Next.js + React.

## Requisitos

- Node.js 18+
- npm

## Configuracion

1. Copiar `.env.example` a `.env.local`.
2. Completar variables de entorno con valores reales.

## Desarrollo

```bash
npm install
npm run dev
```

## Estructura base

- `app/`: rutas y layout de Next.js
- `components/`: componentes de UI y layout
- `lib/`: contexto y utilidades compartidas
- `styles/`: estilos globales y tokens
